# Smart ToDo API

A secure RESTful ToDo API built with FastAPI, MongoDB, and JWT authentication.

## Features
- User registration & login
- JWT authentication
- Password hashing (bcrypt)
- User-specific CRUD for tasks
- MongoDB Atlas integration

## Tech Stack
FastAPI · MongoDB · JWT · Python

## Run Locally
```bash
pip install -r requirements.txt
uvicorn main:app --reload
